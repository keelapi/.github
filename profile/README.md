# Keel

**Decide what your AI can do. Prove what it did.**

Keel is a vendor-neutral authorization and accountability layer for AI agents:
signed permits before an agent acts, independently verifiable cryptographic
evidence after.

| Repo | What it is |
|---|---|
| [keel-verifier](https://github.com/keelapi/keel-verifier) | Independent verifier for signed AI audit exports — `pip install keel-verifier` |
| [keel-permit](https://github.com/keelapi/keel-permit) | Open specification + test vectors for pre-execution AI action authorization records |
| [terraform-provider-keel](https://github.com/keelapi/terraform-provider-keel) | Official Terraform provider — Keel resources as code |
| [keel-go](https://github.com/keelapi/keel-go) | Go SDK for the Keel API |
